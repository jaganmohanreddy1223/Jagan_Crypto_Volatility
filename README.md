## Jagan Crypto Volatility Risk Analyzer
## Project Description

The Crypto Risk Dashboard is an interactive Streamlit-based application that analyzes cryptocurrency price movements and classifies assets based on risk levels using quantitative financial metrics.

It helps users understand crypto volatility, risk exposure, and performance trends through visual analytics.

## Objectives
Analyze cryptocurrency price variations over time
Measure volatility and risk metrics
Classify assets based on risk levels
Visualize insights using interactive dashboards
## Key Features
🔗 Live and historical cryptocurrency data integration
📊 Volatility calculation
📈 Sharpe Ratio analysis
⚠️ Risk classification of assets
📉 Interactive data visualizations
🌐 API-based data fetching (CoinGecko)
📁 Kaggle dataset integration
🛠️ Technical Stack
Programming Language: Python
UI Framework: Streamlit + HTML
Data Processing: Pandas, NumPy
Visualization: Matplotlib, Plotly
API Integration: CoinGecko API
Dataset Source: Kaggle
