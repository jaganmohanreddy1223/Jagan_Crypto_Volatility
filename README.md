# Jagan_Crypto_Volatility
## Description 
The project provides **interactive dashboards** that classify cryptocurrencies into risk categories based on quantitative analysis.
Regarding the volatile changes in Crypto Coins.
## Objectives
- Analysing crypto currency price variations over time.
- Measuring volatility and risk metrics.
- Classifying the assets based on risks.
- Visualising through interactive dashboards.

  ## Features
  - Connects Live and historical data.
  - Volatility Calculation.
  - Sharpe Ratio Analysis.
  - Risk Classification.
  - Interactive visuals and Interface.
  - API data fetching.

## Technical Features
  - Uses Python as core language.
  - Used HTML & Streamlit for UI Development.
  - Mainly focuses on Pandas, Numpy & Matplot Libraries.
  - Uses CoinGecko API.
  - Extracts historical data source from Kaggle.
